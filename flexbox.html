html

HTML, which stands for Hypertext Markup Language, is a standard markup language used to structure and display content on the internet.

c

C is a general-purpose programming language that is known for its efficiency and direct manipulation of hardware and memory, making it particularly suitable for system-level programming.

c++

C++ is a high-level, general-purpose programming language created to extend the C programming language, adding object-oriented (OOP) features.

css

Cascading Style Sheets (CSS) is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML.

java

Java is a high-level, general-purpose, object-oriented programming language designed for building applications that can run on any platform without modification.

java-script

JavaScript is a dynamic computer programming language that is widely used for adding interactive elements to web pages.

python

Python is a high-level, general-purpose programming language designed to emphasize code readability with the use of significant indentation.

swift

Swift is a powerful and intuitive language developed by Apple for iOS, macOS, and other Apple platforms.
