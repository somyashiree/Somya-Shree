html

HTML, which stands for Hypertext Markup Language, is a standard markup language used to structure and display content on the internet.

It defines the content and structure of web pages and is often used in conjunction with technologies like CSS and JavaScript to enhance web page functionality and appearance.

c

C is a general-purpose programming language that was created by Dennis Ritchie at Bell Labs in 1972.

It is known for its efficiency and direct manipulation of hardware and memory, making it particularly suitable for system-level programming and operating systems development, such as Unix and Linux.

c++

C++ is a high-level, general-purpose programming language created by Danish computer scientist Bjarne Stroustrup

in 1979 to extend the C programming language, adding object-oriented (OOP) features.

css

Cascading Style Sheets (CSS) is a style sheet language used for specifying the presentation and styling of a document

written in a markup language such as HTML or XML, including XML dialects like SVG, MathML, or XHTML.

java

Java is a high-level, general-purpose, object-oriented programming language designed for building applications

that can run on any platform without modification, adhering to the principle of "write once, run anywhere" (WORA).

java-script

JavaScript is a dynamic computer programming language that is widely used for adding interactive elements to web pages.

It was first introduced on December 4, 1995, and was initially known as LiveScript, but its name was changed to JavaScript by Netscape due to the popularity of Java at the time.

python

Python is a high-level, general-purpose programming language designed to emphasize code readability with the use of significant indentation.

It supports multiple programming paradigms, including structured (particularly procedural), object-oriented, and functional programming. Python is dynamically type-checked and garbage-collected, making it a flexible and versatile language for various applications such as web development, software development, task automation, data analysis, and data visualization.

swift

Swift is a powerful and intuitive language developed by Apple for iOS, macOS, and other Apple platforms.

It is designed to be fast, modern, safe, and enjoyable to write. Swift supports modern, straightforward syntax for declaring new types and provides default values for instance properties and custom initializers.
